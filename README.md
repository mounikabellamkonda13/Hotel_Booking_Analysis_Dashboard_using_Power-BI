# **1. Project Title and Description**

## **Hotel Booking Analysis Dashboard | Power BI Project**

This project focuses on analyzing hotel booking data to uncover meaningful insights related to booking trends, customer behavior, revenue performance, and cancellation patterns. An interactive Power BI dashboard was created to help hotel management understand business performance and make better data-driven decisions.

---

# **2. Short Description / Purpose**

The purpose of this project is to transform raw hotel booking data into a clear and interactive dashboard that helps stakeholders monitor key hotel performance metrics, identify customer booking patterns, reduce cancellations, and improve revenue strategies.

---

# **3. Tech Stack**

* **Power BI Desktop** – Used to create the interactive dashboard and visual reports
* **Power Query** – Used for data cleaning, transformation, and preparation
* **DAX (Data Analysis Expressions)** – Used to create calculated columns, measures, and KPIs
* **Data Modeling** – Used to build relationships between tables for accurate analysis
* **File Format** – `.pbix` for development and `.png` for dashboard preview

---

# **4. Data Source**

The dataset used in this project is the **Hotel Booking Dataset**, which contains booking information for both **City Hotel** and **Resort Hotel**.

**Source:** Kaggle
**Link:** [https://www.kaggle.com/datasets/mojtaba142/hotel-booking](https://www.kaggle.com/datasets/mojtaba142/hotel-booking)

### **Dataset Details**

* **Total Records:** 119,390
* **Total Features:** 32+ columns

The dataset includes details such as booking dates, guest information, hotel type, stay duration, booking channels, customer type, and reservation status.

---

# **5. Feature Highlights**

## **Business Problem**

Hotels receive a large number of bookings from different channels, but they often face several business challenges such as:

* High cancellation rates
* Changing booking patterns
* Difficulty in understanding customer behavior
* Revenue fluctuations across hotel types and channels

These issues make it difficult for hotel management to optimize operations and improve profitability.

---

## **Goal of the Dashboard**

The main goal of this dashboard is to analyze hotel booking data and provide clear business insights that help management:

* Understand booking trends
* Track revenue performance
* Analyze cancellation behavior
* Explore customer patterns
* Support better business decision-making

---

## **Walkthrough of Key Visuals**

### **1. Overview Page**

This page provides a high-level summary of hotel performance using important KPIs such as:

* Total Bookings
* Total Revenue
* ADR (Average Daily Rate)
* Cancellation Rate

It helps users quickly understand the overall business performance.

### **2. Revenue Analysis**

This page focuses on hotel revenue performance and includes visuals related to:

* Revenue by hotel type
* Revenue trends over time
* ADR comparison
* Revenue contribution by booking channels

It helps identify which hotel type and channel generate more revenue.

### **3. Booking Behavior**

This page analyzes customer booking patterns through visuals such as:

* Monthly booking trends
* Lead time distribution
* Guest group distribution
* Stay duration analysis

It helps understand how customers book and how long they stay.

### **4. Cancellation Insights**

This page highlights cancellation-related patterns using visuals such as:

* Cancellation rate by hotel type
* Cancellation by booking channel
* Lead time vs cancellation behavior
* Reservation status breakdown

It helps identify factors that influence cancellations.

### **5. Customer Insights**

This page focuses on customer-related analysis and includes:

* Customer type distribution
* Repeated vs new guests
* Guest composition
* Top booking countries

It helps understand customer segments and booking sources.

---

## **Business Impact and Insights**

### **Key Insights**

* City Hotels receive a higher number of bookings compared to Resort Hotels
* A large portion of bookings comes through **Online Travel Agents (OTA)**
* The overall cancellation rate is relatively high and impacts revenue
* Most bookings are made for **short stays (1–3 nights)**
* A small number of countries contribute to the majority of hotel bookings

### **Business Impact**

This dashboard helps hotel management to:

* Monitor overall booking and revenue performance
* Identify high-risk cancellation patterns
* Understand customer booking behavior
* Improve pricing and forecasting strategies
* Support operational and strategic business decisions
